## Миссия

Создать «цифровой мозг» нации — Ситуационный центр при Президенте КР.
Платформа превращает сайт Президента из информационной витрины в **инструмент реального управления государством**.

---

## Архитектура (из концепции)

### Слой 1: Суверенный ИИ
- Локальные LLM (Llama/Mistral) дообученные на кыргызском законодательстве
- Государственный ЦОД + G-Cloud на GPU-кластерах
- Система «Тундук» — 50+ госбаз данных

### Слой 2: Двойной интерфейс
- **Human UI** — дашборд Президента + портал граждан
- **Machine-Readable API** — JSON-LD для AI-агентов министерств

### Слой 3: Ситуационный центр
- Digital Twin экономики КР
- KPI мониторинг министерств в реальном времени
- Предиктивная аналитика (Policy Rehearsals)

### Слой 4: Децентрализация
- GovDAO — блокчейн-логирование указов
- LoRa-Mesh сети (оффлайн-правительство)
- Edge AI для регионов

## Текущий прогресс

### ✅ Готово
- Панель управления (React/TypeScript/Canvas) — `panel/`
- 6 министерств с живой анимацией агентов
- 7 AI-агентов: САДЫР, АК-МАРАЛ, ЖЕНИШ, БЕРИК, НУРЗАТ, АЙГЕРИМ, ТИЛЕК
- Список всех 14 министерств в сайдбаре
- Журнал событий, инфраструктурная панель
- Герб Президента КР интегрирован

### 🔄 В разработке
- Подключение реальных OpenClaw агентов к министерствам
- KPI мониторинг на основе реальных данных
- MCP интеграция с «Тундук»

### ⏳ Планируется
- Digital Twin экономики КР (Mesa/симуляции)
- JSON-LD machine-readable API
- Блокчейн-логирование указов
- LoRa-Mesh fallback сеть
