### 2. TypeScript в .mjs файлах (из ERRORS.md)
**Pattern:** `SyntaxError: Unexpected token ':'` при добавлении TypeScript аннотаций в `.mjs`  
**Root cause:** Путаница между `.ts` и `.mjs` форматами  
**Fix applied:** ✅ Задокументировано в ERRORS.md (2026-03-07)  
**Recommendation:** Добавить ESLint rule для предотвращения TS в .mjs

### 3. Двойной Cloudflare Tunnel процесс (из ERRORS.md)
**Pattern:** Config-based tunnel игнорируется когда token-based tunnel уже запущен  
**Root cause:** Два cloudflared процесса с разными источниками конфига  
**Fix applied:** ✅ Kill token-based, оставить только config-based  
**Recommendation:** Добавить в LaunchAgent check: `pkill -f "tunnel run --token"` перед стартом

### 4. SSH Gateway Restart drops (из ERRORS.md)
**Pattern:** `exit code 255` при `ssh host "systemctl restart openclaw-gateway"`  
**Root cause:** SSH соединение умирает вместе с gateway  
**Fix applied:** ✅ Использовать bash heredoc или внешнюю проверку  
**Recommendation:** Всегда использовать `nohup` + check via secondary connection

### 5. PM2 скрывает Node.js ошибки (косвенный паттерн)
**Pattern:** PM2 показывает `status: online` когда процесс стартовал, но упал из-за SyntaxError  
**Root cause:** PM2 restarts быстро, не логирует причину crash чётко  
**Fix applied:** Использовать `pm2 logs --lines 20 --nostream` после каждого restart  
**Recommendation:** Добавить `node --check file.mjs` в pre-deploy скрипт
