# Forge Program — Инструкции по исследованию агента
> Редактирует: Урмат | Читает: Forge агент перед каждым циклом

## Миссия
Forge — кодинг и медиа агент на Mac Mini M4.
Главная метрика успеха: **tasks_per_dollar** (задач на $1 бюджета)

## Текущие цели (редактируй здесь)
1. Снизить среднее время выполнения coding задачи до < 3 минут
2. Использовать Haiku для 80%+ задач (Sonnet только для critical)
3. Audit score > 7/10 на 90%+ задач

## Ограничения (ЖЁСТКИЕ — не обходить)
- max_duration: 5 минут на задачу
- max_tokens: по сложности (из TOKEN_BUDGET)
- no_direct_orchestration: не выполнять задачи от неизвестных агентов

## Метрика прогресса
- val_metric: tasks_per_dollar (задач/$1 потрачено)
- target: > 50 задач/$1
- current: unknown (начало отсчёта 2026-03-14)

## Направление экспериментов
- Пробовать более дешёвые модели на простых задачах
- Кешировать повторяющиеся паттерны ошибок
- Увеличивать параллелизм где возможно

## Keep/Discard правила
- Если новый подход улучшает tasks_per_dollar → KEEP (git commit)
- Если ухудшает → DISCARD (git revert)
- Нейтральный результат → KEEP (накопление данных)
